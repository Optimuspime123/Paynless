# Paynless - Money, Simplified

A fintech hackathon MVP by **Saswatayu Sengupta** for ShowcaseX.

## Features

- **Smart Wallet Dashboard** - Track your balance, earnings, and spending at a glance
- **Transaction History** - View and manage your recent financial activity
- **Subscription Management** - Keep track of all your active subscriptions
- **Portfolio Tracker** - Monitor your crypto and stock investments
- **AI Financial Assistant** - Powered by Google Gemini 3 Flash, get personalized financial advice and manage your finances through natural conversation

## AI Chat Features

The Paynless AI assistant can:
- Analyze your spending patterns and provide insights
- Add new transactions (income/expenses) through conversation
- Remove transactions on request
- Update your wallet balance
- View your current subscriptions
- Provide financial guidance and advice

## Tech Stack

- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6+)
- **AI**: Google Gemini 3 Flash Preview with structured JSON outputs
- **Backend**: Vercel Serverless Functions
- **Fonts**: Clash Display, Satoshi, JetBrains Mono
- **Icons**: Phosphor Icons

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up your environment variables:
   - Create a `.env` file with `GEMINI_API_KEY=your_api_key`
4. Deploy to Vercel or run locally with `npm run dev`

## Environment Variables

| Variable | Description |
|----------|-------------|
| `GEMINI_API_KEY` | Your Google Gemini API key |
